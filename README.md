# imgur-discord-command-cfworker

Discord slash-command for uploading content from an url to [Imgur](https://imgur.com).   
Deployable on Cloudflare Workers.

## Configuration

| Worker Secrets               |
|------------------------------|
| `IMGUR_CLIENT_ID`            |
| `IMGUR_CLIENT_SECRET`        |
| `IMGUR_REFRESH_TOKEN`        |
| `DISCORD_APPLICATION_ID`     |
| `DISCORD_APPLICATION_SECRET` |
| `DISCORD_PUBLIC_KEY`         |
